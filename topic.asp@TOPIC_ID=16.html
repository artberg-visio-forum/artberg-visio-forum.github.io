Скорее всего сработает вот такой способ (далее цитата из FAQ):
ВАРИАНТ 1 можно попробовать обойтись подстановкой шрифтов: - найти файл Win.ini, найти в нем секцию [FontSubstitutes] - добавить туда строчку типа Arial,0=Arial,204 (Это если вы используете шрифт Arial. Для других шрифтов нужно по аналогии добавить свои строчки). После этого Windows будет подсовывать Word'у в нужный момент соответствующую часть кодовой таблицы.
