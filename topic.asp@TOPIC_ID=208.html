Для создания шаблона по умолчанию, он должен называться Blank Drawing.vst и лежать в папке Solutions. Откройте его в режиме редактирования, заполните трафаретами и все. При создании нового документа, Visio будет использовать именно его.

Если этого не произойдет, то следует изменить в реестре по адресу HKEY_CURRENT_USER\Software\Visio\Visio 2000\application строковый параметр "NewDocCreateMode" на значение "2"
Либо для Visio 2002
следует изменить в реестре по адресу HKEY_CURRENT_USER\Software\Microsoft\Visio\application строковый параметр "NewDocCreateMode" на значение "2"
