help по F1 в ячейке:
Определяет, отображается ли разметка в окне рисунка.


Значение Описание
TRUE
На рисунке отображается разметка.

FALSE
Разметка не отображается (по умолчанию).


Замечания
При включенном отслеживании разметки (значение TRUE в ячейке AddMarkup) ячейке ViewMarkup автоматически присваивается значение TRUE, которое остается даже после того, как отслеживание разметки отключено (значение FALSE в ячейке AddMarkup). Когда ячейке AddMarkup присвоено значение TRUE, значение в ячейке ViewMarkup игнорируется.

Ячейке ViewMarkup также присваивается значение TRUE при добавлении комментариев к рисунку (независимо от того, включено или нет отслеживание разметки), и это является необходимым условием для возможности просматривать комментарии.

Эта ячейка соответствует команде Разметка в меню Вид.
