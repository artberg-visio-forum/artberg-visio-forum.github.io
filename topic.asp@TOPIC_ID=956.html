1. Что-то типа словаря есть не на форуме, а на сайте. См., http://visio.artberg.ru/manual/Text/T6.htm
2. В Visio есть несколько поставляемых Add-On, ориентированных на эту задачу. Искать их в Tools / Macros / Visio Extras (с именами, начинающимися на Database и Reports). Ну, а если их возможностей не хватает, тогда писать свой Add-On.
