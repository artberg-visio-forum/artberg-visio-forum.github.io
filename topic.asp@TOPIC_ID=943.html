Я вижу 2 способа.
1. File - Page Setup - Page Size. Далее установить pre-defined size A1.
Но, я так понимаю, что рисунки в визио уже нарисованы, и перерисовывать времени нет. Тогда, скорее подойдет способ 2.
2. File - Page Setup - Print Setup. В секции Print zoom установить Adjust to 300%.

Это для английской версии визио. Для русской не трудно будет догадаться.

Странно. У меня visio 2003 Professional Eng.
И нормально вручную установился масштаб 300%.

Есть, конечно, вариант...
Только он ближе к 1-му способу.
Установить размер страницы А1, а все что было на маленьком листе скопировать и сделать специальную вставку: Edit - Paste special... - Picture (Enhanced metafile). После чего попробовать вставленный рисунок растянуть на весь лист А1.
