Продолжаем ретроспективу перечисления глюков визио 2002, 2003:
Сцена первая: Microsoft малацца, афтар пиши исчо
по шагам:
1. новый документ Building Plan -> Electrical and Telecom
2. Add-on -> Building Plan -> Enable Space Plan (наблюдаем Space Explorer => адон запустился)
3. Бросаем на лист шэйп "Space" (3х3 m)
4. Бросаем в центр поля "Space" Socet outlet
5. для нашего Socet в меню Outlet Plane -> Assign Category -> Asset
6. сдвинули чутку наш Socet Outlet чтобы Space Explorer его "схватил"
--- итог первой сцены опуса
в Space Explorer в закладке Spaces
Drawing1
- Page1
- Space
Asset
это все показывает что внутри поля Space содержицца Asset (Socet Outlet), вах афтар жжот, для спецификаций расчетов итп, это проста супер
Сцена вторая: -Тоесть как, "пошел на %*№"!!!!
сдвигаем наш сокет оутлет в левый нижний угол(где хотелось бы чтобы стояла розетка) Space-а до тех пор пока он не "выпадет" из Space-а с точки зрения Space Explorer-а
в тоже время визуально наш Asset находится внутри поля Space-а
ps для продолжения экспиримента наш Asset должен стоять "на грани" выпадания из Space-а

Сцена третья: Входит и выходит
Дано: Asset внутри Space(визуально) вне спейса для Space Explorer
Решение: поворт Asset в любом направлении на 90 градусов

теже самые эксперименты можно проводит над шэйпами типа Switches, я не стал продолжать над собой издеваться, мне хватило розеток и выключателей
