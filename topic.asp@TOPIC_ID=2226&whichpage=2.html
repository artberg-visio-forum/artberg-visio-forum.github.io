Действительно, в Visio 2007 либо выбросили, либо куда-то спрятали нужное окошко :) А может у меня версия какая кривая...
Сразу даже не могу сориентироваться, где лучше искать ID... Пока вижу только два варианта:
1. Селектировать шейп и выполнить вот такой макрос
Sub ttt()
Debug.Print ActiveWindow.Selection(1).ID
End Sub
Он выведет в окошко Immediate ID селектированного шейпа. Число.
2. Выполнить меню Data / Reports, создать отчет для селектированных шейпов, поставить в нем галочку NameID и потом запустить отчет. Там NameID тоже показывается нормально.
А "правильный" путь скажу попозже, когда найду :)

Разобрался.
Оказывается в версии 2007 Microsoft чуть прикрыла от неопытных пользователей совершенно не нужные последним сведения. Ранее свободно наблюдаемая форма стала видна только в developer mode.
Желающим наблюдать в меню строчку "Специальные..." придется теперь:
- выбрать меню Tools / Options...
- выбрать вкладку Advanced
- В панели Advanced options выставить галочку Run in developer mode
- нажать OK.
После этого можно, как и раньше, селектировать шейп и давить "Формат". Строчка Special в меню будет.
Кстати, developer mode прикрывает не только этот пункт меню...
