Могу посоветовать следующий вариант, любую книгу по визуал бейсику, лучше вместе с VBA, естественно по визио там ни чего не будет, но это не беда, принцип работы с визио из бейсика такой-же как и с другими приложениями офиса, по поводу access, на мой взгляд для разработки новых баз трудна, если вам не надо делать сложные базы данных рекомендую воспользоваться FreeWare библиотекой Максима Тищенко MODBF.dll, к сожалению не помню откуда скачал, по мыло могу послать, есть хороший и понятный пример и справка. теперь на счет визио, если надо делать в VBA, то предлагаю анализировать записанные макросы в визио 2003, если из VBА надо переделать в VB, то просто макрос незначительно переделывается по инструкциям на этом сайте, а access по моему больше не поддерживается (не развивается), другое дело если у вас уже есть документы в ассess, рекомендую посетить странички по работе с VB. удачи
