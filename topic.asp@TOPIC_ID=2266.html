Скорее всего можно.
Многие из нужных параметров передаются через свойства документа. Примерно так
Set docObj = appVisio.Documents(1)
'Прием всех параметров печати принтера
sTP = docObj.Printer
PaperSize = docObj.PaperSize
Orient = docObj.PrintLandscape
CenterH = docObj.PrintCenteredH
CenterV = docObj.PrintCenteredV
FitP = docObj.PrintFitOnPages
FitH = docObj.PrintPagesAcross
FitV = docObj.PrintPagesDown
Pleft = docObj.LeftMargin * 25.4
Pright = docObj.RightMargin * 25.4
Ptop = docObj.TopMargin * 25.4
Pbottom = docObj.BottomMargin * 25.4
Естественно, их можно как читать, так и писать.
