Поправка: белый лист есть выводимая на печать область, размер бумаги принтера это более другая фишка. Поменять можно так: PageSetup->PageSize->CustomSize-> 1mm x 1 mm. ИМХО, исходя из этой же логики, убрать эту штуку не представляется возможным. Советую просто изменить цвет (правда настройка для всего Visio) – Tools->Options->View->Color Settings->DrawingPage.

Данные параметры, как впрочем все остальные, относящиеся к цветовому оформлению вы можете изменить, либо в программе, либо через реестр.
Через программу. Сервис>Параметры>Настройка цветов>Страница.
Либо через реестр.
Для пользователей Visio 2000.
Адрес: HKEY_CURRENT_USER\Software\Visio\Visio 2000\colors
Для пользователей Visio 2002
Адрес: HKEY_CURRENT_USER\Software\Microsoft\Visio\colors
Если копаться в реестре сложно, то можете просто скачать файл
www.prodigitall.narod.ru/stencils/color4.reg
Если данные настройки не подходят, то откройте с помощью блокнота и измените. Все цвета заносятся в виде "RRR GGG BBB".
Останется только изменить стили в шаблоне и все.
