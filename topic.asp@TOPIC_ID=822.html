А макросы Вас не смущают?
С помощью макроса делается довольно легко. Макрос можно запускать из контекстного меню. Он активизирует другой лист, селектирует нужный шейп...
Только передавать каждый раз свои параметры запуска для разных шейпов довольно неприятно. Может лучше будет прописать какую-нибудь таблицу связей. Тогда макросу можно передавать только имя шейпа, на котором его запустили, а остальные данные он возьмет из таблицы.
А без макроса - переходы делаются, а вот селектирование - затруднительно.
