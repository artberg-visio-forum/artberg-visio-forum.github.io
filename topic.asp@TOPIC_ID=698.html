Делаете это так:
1 Запускаете делфи
2 Кликаете на меню Component
3 Выбираете Import ActiveX Control
4 Из появившегося списка выбираете Microsoft Visio Drawing Control Type Library
5 Нажимаете Install
6 В указанной вкладке должен появится компонент Visio, который если кинуть на форму можно юзать как само приложение Visio
