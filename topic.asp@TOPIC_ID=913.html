Собственная штриховка делается путем создания и использования своих образцов заливки (Custom Patterns).
Образцы заливки аналогичны мастер-шейпам. Можно создать сколько угодно своих образцов, сохранить их и использовать при необходимости. Сохраняются они, как и другие мастер-шейпы, в трафарете, но есть особенность - их там не видно. Пользователю они доступны либо через меню выбора типа заливки, либо через окно Drawing Explorer.
Сделать так, чтобы штриховка строилась под любым углом будет тяжело. Проще создать множество образцов с фиксированными углами. Главное, что здесь должно учитываться - при совмещении двух соседних образцов линии одного образца должны являться продолжением линий другого.
Создание образца
1. Открываем новый рисунок Visio (не обязательно новый).
2. Открываем окно View / Drawing Explorer.
3. Щелкаем правой кнопкой на Fill Patterns и выбираем New Pattern.
4. В появившемся окне даем штриховке имя, например, 30 град. Отмечаем Fill Pattern и в качестве типа отображения выбираем самый левый (размножающиеся образцы). Жмем OK.
Образец создан. Пока он пустой.
Редактирование образца
1. В окне Drawing Explorer Window щелкаем правой кнопкой на образце и выбираем пункт Edit Pattern. Открывается окно редактирования образца. Оно очень похоже на окно рисунка.
2. Рисуем в нем несколько параллельных линий под нужным углом с учетом вышесказанного замечания о совмещении.
3. Рекомендуется весь образец размещать в одной секции геометрии. Для нас это выступает как требование - после создания рисунка выполнить операцию Join.
4. Закрываем окно редактирования и отвечаем Yes на запрос подтверждения.
Использование образца
Образец применяется через меню Format / Fill. В окне Fill нужно заглянуть в самый конец списка Pattern, где и будут перечислены все пользовательские образцы заливки.
Копирование и использование в других рисунках
Лучший путь для использования пользовательских штриховок в других рисунках - создать на основе нашего пустого рисунка шаблон, а в дальнейшем создавать рисунки на основе этого шаблона. В этом случае нужные образцы будут всегда находиться в трафарете документа (Document Stencil).
Если штриховку нужно применить в произвольном документе, то можно поступить следующим образом:
- открыть дополнительно наш документ, в котором есть штриховки;
- в окне Drawing Explorer Window найти нужную штриховку и выполнить Copy;
- в нужном документе выполнить Paste.
При этом образец штриховки появится на рисунке в виде шейпа. Но одновременно он в виде мастер-шейпа попадет в трафарет документа. Шейп с рисунка можно убрать, а нужный мастер в трафарете останется и штриховку уже можно будет использовать в данном документе.
