Вроде уже спрашивали такое...
Ответ: The numeric values in this cell map to constants (prefixed with DMPAPER) defined for paper selections in the

Microsoft Windows wingdi.h file
То есть, если открыть файл wingdi.h, то можно найти и список форматов.
Вот я нашел, посмотрел. Получил вот это:
