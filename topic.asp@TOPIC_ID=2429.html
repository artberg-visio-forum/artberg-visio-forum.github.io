нужно, чтобы открывалась та страница, на которой нужная фигура, камера бы на нее наезжала так, чтобы фигура оказывалась в центре рабочего окна и немного приближался вид, типа zoom.

разобрался. сделал через ф-ю SetViewRect

Dim sh As Visio.Shape

Set sh = ActivePage.Shapes("Sheet.1")
Application.ActiveWindow.DeselectAll
Application.ActiveWindow.Select sh, visSelect
Application.ActiveWindow.SetViewRect _
sh.Cells("PinX") - 2, sh.Cells("PinY") + 2, 4, 4

если не нужно делать укрупнение плана, а просто сделать выделенный шейп в центре экрана…
то в 2010 можно сделать немного короче :)

Dim sh As Visio.Shape
Set sh = ActivePage.Shapes.ItemFromID(1)
Application.ActiveWindow.CenterViewOnShape sh, visCenterViewSelectShape
